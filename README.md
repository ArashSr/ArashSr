I build systems that scale. Years deep in distributed systems, networking protocols, and the infrastructure that makes deep learning actually run at scale — now I'm channeling that energy into open source.

Right now I'm focused on the unglamorous parts that matter: reliable error detection in data transfers, hybrid cloud, and squeezing performance out of large AI workloads. The kind of work where a small efficiency gain quietly saves thousands of GPU-hours.

If you're into distributed systems, networking protocols, or scalable AI infrastructure — we should talk.

⚡ Fun fact: I'm fluent in three languages and a lifelong open-source believer.
